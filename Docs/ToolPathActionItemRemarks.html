The following content is placed in the <remarks> section of the comments for ToolPathActionItem.

Note that there are some interrelated properties available on this class, many of which are exclusive to one or more others.

The following sections enumerate the contexts in which the properties are used, as well as their order of precedence when considering exclusivity.

Contexts

  • Dimensioning. Following are the general dimensioning properties used in this version.
    • Angle. The angle at which to point from the reference. In this version, for absolute angles, the angular system is based upon a right-facing ray whose angular value increases in a counterclockwise directon. When used as a deviation from a current movement, a positive number will increase to the left, while a negative number will increase to the right of that movement.
  • Elliptical Cuts. Some properties are only used when making elliptical movements.
    • Diameter. The singular value of the X and Y diameters for the elliptical pattern, resulting in a circle.
    • Radius. The singular value of the X and Y radii for the elliptical pattern, resulting in a circle.
    • Diameter X. The X diameter of the elliptical pattern.
    • Diameter Y. The Y diameter of the elliptical pattern.
    • Radius X. The X radius of the elliptical pattern.
    • Radius Y. The Y radius of the elliptical pattern.
  • Positioning. Different types of positioning values are allowed, depending upon the type of cut to be made.
    • X and Y represent offsets of a shape from its corner, center, or other scalar position. They are used for drill points, lines, paths, rectangles, curves, and elliptical cuts.
    • X1, Y1, X2, Y2 are all used when the shape is defined by two points.
    • Z, in this version, is a special case because the head has only a depth measurement (eh-hm, please post related Issues on the GitHub site if you want to see 2.5D carving capabilities before I otherwise get around to it...).

Cut Types

Following are the currently supported cutting styles, associated with the names of the properties they align with.

Cut TypePropertiesDescription
DrawCircleCenterDiameter X, Y, Diameter Draw a circle using a center reference point and diameter.
DrawCircleCenterRadius X, Y, Radius Draw a circle using a center reference point and radius.
DrawCircleDiameter X, Y, Diameter Draw a circle using corner X, Y references and diameter.
DrawCircleRadius X, Y, Radius Draw a circle using corner X, Y references and radius.
DrawEllipseCenterDiameterXY X, Y, DiameterX, DiameterY Draw an ellipse using a center reference point and independent diameters.
DrawEllipseCenterRadiusXY X, Y, RadiusX, RadiusY Draw an ellipse using a center reference point and independent radii.
DrawEllipseDiameterXY X, Y, DiameterX, DiameterY Draw an ellipse using corner X, Y references and independent diameters.
DrawEllipseRadiusXY X, Y, RadiusX, RadiusY Draw an ellipse using corner X, Y references and independent radii.
DrawLineAngleLength X, Y, Angle, Length Draw a line using a point, an angle, and a length.
DrawLineLengthWidth X, Y, Length, Width Draw a line using a point, a length, and a width.
DrawLineXY X1, Y1, X2, Y2 Draw a line using two points.
DrawPath PathData Draw the path specified in the PathData property.
DrawRectangleLengthWidth X, Y, Length, Width Draw a rectangle using a corner, length, and width.
DrawRectangleXY X1, Y1, X2, Y2 Draw a rectangle using two corner points.
FillCircleCenterDiameter X, Y, Diameter Fill a circle using the center reference and a diameter.
FillCircleCenterRadius X, Y, Radius Fill a circle using its center reference and a radius.
FillCircleDiameter X, Y, Diameter Fill a circle using its corner reference and a diameter.
FillCircleRadius X, Y, Radius Fill a circle using its corner reference and a radius.
FillEllipseCenterDiameterXY X, Y, DiameterX, DiameterY Fill an ellipse using a center reference point and independent diameter values.
FillEllipseCenterRadiusXY X, Y, RadiusX, RadiusY Fill an ellipse using a center reference point and independent radii.
FillEllipseDiameterXY X, Y, DiameterX, DiameterY Fill an ellipse using a corner point and independent diameter values.
FillEllipseRadiusXY X, Y, RadiusX, RadiusY Fill an ellipse using a corner point and independent radii.
FillPath PathData Fill the path specified in the PathData property.
FillRectangleLengthWidth X, Y, Length, Width Fill a rectangle using one corner, width, and height.
FillRectangleXY X1, Y1, X2, Y2 Fill a rectangle using two corners.
MoveAngleLength X, Y, Angle, Length Move the tool, without cutting, at an angle, by a specified length.
MoveXY X, Y Move the tool, without cutting, to the specified coordinate.
PointXY X, Y Drill at a point represented by the X and Y coordinates.
